# Tetris Game Rules

## Board

- Size: 10 x 20 grid

## Tetromino Types

- I
- O
- T
- S
- Z
- J
- L

## Controls

Move Left : ArrowLeft Move Right : ArrowRight Soft Drop : ArrowDown Hard Drop : Space Rotate : ArrowUp Hold Block : Shift

## Game Flow

1. 블록 생성
2. 플레이어 이동 / 회전
3. 바닥 또는 블록 충돌 시 고정
4. 라인 검사
5. 라인 제거
6. 다음 블록 생성

## Line Clear Score

1 line : 100 2 lines : 300 3 lines : 500 4 lines : 800 (Tetris)

## Game Over

- 블록 생성 위치에 이미 블록 존재
